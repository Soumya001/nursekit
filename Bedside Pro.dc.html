How to use
1 Enter values exactly as written on the order — unit for unit. Do not convert in your head before entering.
2 Read the result and the working line below it. If the working doesn't match how you'd do it by hand, stop and recheck.
3 Cross-check the result against the prescriber's order. If they differ, the order is the authority — query the order, don't adjust the dose.
4 For high-risk calculations (titration, chemotherapy, paediatric), have a second qualified clinician independently verify the result before giving.
5 If the result or any warning feels wrong — trust that instinct. Call pharmacy or the prescriber before proceeding.
When NOT to use this app
blockYou are unfamiliar with the drug, route, or indication — look it up first.
blockThe patient is deteriorating — escalate immediately rather than calculating.
blockYour policy mandates a pharmacist check — this app does not replace that requirement.
blockComplex titration with multi-step adjustments — use the pump's validated drug library instead.
blockThe order is unclear or ambiguous — clarify with the prescriber before calculating.
